# vuePages

## 下载依赖包
```
npm install
```

### 本地运行
```
npm run serve
```

### 打包
```
npm run build
```

### 项目说明
通过改造vue-cli3的脚手架利用多页实现按需打包!在同一脚手架中可打包出多个项目!
